# SHU-Utility-MiniProgram 

本项目是一个基于微信小程序原生框架开发的综合性应用，包含了**上海大学《小程序开发》课程作业**以及个人自研的实用工具集。

## 📑 目录说明

- `/pages/homework/`：存放上海大学小程序课程的所有实验作业（Lab 1-3）
- `/pages/tools/`：存放个人开发的实用小工具（抽奖转盘、排卦、课表等）
- `/cloudfunctions/`：存放微信云开发相关的后端函数逻辑

## 🛠 功能介绍

### 1. 上海大学课程作业 (Coursework)
严格按照课程要求完成，主要展示以下技术点：
* **数据绑定与交互**：实现复杂页面的逻辑渲染。
* **网络请求与API调用**：对接校内或开放接口数据。
* **云数据库操作**：使用 JSON 数据库进行数据的持久化存储（CRUD）。

### 2. 集成小工具 (Built-in Tools)
* **美食旋转盘**：通过随机算法解决“今天吃什么”的选择难题。
* **周易六十四卦**：基于数字起卦逻辑的传统文化趣味工具。
* **学生课表直通车**：针对校内排课习惯优化的课程管理界面。

## 🚀 技术栈
- **前端**：WXML / WXSS / JavaScript
- **后端**：微信云开发 (WeChat Cloud Development)
- **数据库**：云开发内置 NoSQL 数据库

## 💻 如何在本地运行
1. 使用 **微信开发者工具** 导入本项目。
2. 修改 `project.config.json` 中的 `appid` 为你自己的小程序 ID。
3. 开通**云开发环境**，并在 `app.js` 中初始化你的环境 ID：
   ```javascript
   wx.cloud.init({
     env: 'your-env-id', // 替换为你的云环境ID
     traceUser: true,
   })
