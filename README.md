# Says-Kotowari-CCB
> **DOCUMENTATION** | 说的道理测测版 · 上海大学移动应用设计与开发课程作业

---

本项目是一个基于微信小程序原生框架开发的综合性应用，包含了**上海大学《移动应用设计与开发》课程作业**以及个人自研的实用工具集。

## 📑 目录说明

- `/pages/homework/`：存放上海大学移动应用设计与开发课程的所有实验作业
- `/pages/tools/`：存放个人开发的实用小工具（嘉然转盘、每日签到、共享课表等）
- `/cloudfunctions/`：存放微信云开发相关的后端函数逻辑

## 🛠 功能介绍

### 1. 上海大学课程作业 (Coursework)
##  项目背景
本项目旨在将课堂所学的微信小程序开发技术应用于实际场景。通过完成课程规定的实验任务。

##  课程作业说明 (SHU Assignments)
本部分严格遵循教学大纲要求，存放于 `/miniprogram/pages/` 目录下，文件夹名称为3_2到5_9的所有文件。

### 2. 集成小工具 (Built-in Tools)
* **嘉然转盘**：通过随机算法解决“今天吃什么”的选择难题。
* **每日签到**：基于数字起卦逻辑的传统文化趣味工具。
* **共享课表**：针对校内排课习惯优化的课程管理界面。

## 🚀 技术栈
- **前端**：WXML / WXSS / JavaScript
- **后端**：微信云开发 (WeChat Cloud Development)
- **数据库**：云开发内置 NoSQL 数据库

## 💻 如何在本地运行
1. 使用 **微信开发者工具** 导入本项目。
2. 修改 `project.config.json` 中的 `appid` 为你自己的小程序 ID。
3. 开通**云开发环境**，并在 `app.js` 中初始化你的环境 ID：
   ```javascript
   wx.cloud.init({
     env: 'your-env-id', // 替换为你的云环境ID
     traceUser: true,
   })
